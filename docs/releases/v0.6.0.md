# OpenWind-AU v0.6.0 Release Notes

OpenWind-AU v0.6.0 adds the Terrain Category Evidence Engine and prepares the project for broader
public review.

## Highlights

- Added 8-direction terrain category evidence summaries.
- Added built-up, vegetation, and open-terrain coverage metrics.
- Added obstruction height statistics, density, spacing, directional fetch, and shielding
  confidence evidence.
- Added separate evidence score components for open exposure, vegetation, urban density, and
  obstruction height.
- Added qualified suggested terrain category ranges for engineering review.
- Added terrain category evidence map, report, UI section, and API endpoints.
- Added representative validation examples for coastal open terrain, suburban housing, dense
  suburban terrain, industrial estate terrain, CBD-like terrain, and rural vegetation.
- Updated release documentation, maturity warnings, templates, and reviewer guidance.

## Safety And Scope

OpenWind-AU is an early-stage engineering support tool. It does not produce certified design
values.

This release does not:

- assign a final AS/NZS 1170.2 terrain category;
- calculate final `Mz,cat` design values;
- claim AS/NZS compliance;
- calculate design wind pressures;
- replace competent engineering judgement.

## Known Limitations

- Public DEM and public footprint data may be incomplete or unsuitable for a specific project.
- DSM-DTM height enrichment depends on externally supplied raster quality and alignment.
- Terrain category evidence is a transparent review aid, not a final code output.
