# Demo Project

This folder contains a small synthetic demo project for exercising OpenWind-AU workflows without
using private project data.

Files:

- `site_request.json` - sample site-analysis and terrain-category request payload.
- `reviewed_obstructions.csv` - sample reviewed obstruction heights for import.
- `sample_footprints.json` - synthetic footprint records for developer tests and examples.

The data is illustrative only. It is not a real project and must not be used for design.
