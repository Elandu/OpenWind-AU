1. Site Inputs

Preliminary workflow only. The app does not calculate final design pressures and does not certify AS/NZS 1170.2 compliance.

Run the assessment to display resolved site inputs.

2. Wind Region / Regional Wind Speed, VR

Displays wind region, ARI / importance level, source reference, and selected VR.

3. Wind Direction Multiplier, Md

Displays direction, selected Md, source reference, and review status.

4. Terrain Category / Mz,cat

Displays Recommended TC, Final TC, Recommended Mz,cat, Final Mz,cat, and review status.

5. Shielding Multiplier, Ms

Displays Recommended Ms, Final Ms, review status, and shielding calculation evidence.

6. Topographic Multiplier, Mt

Displays Recommended Mt, Final Mt, review status, and topographic assessment evidence.

7. Site Wind Speed, Vsit,b

Vsit,b = VR x Md x Mz,cat x Ms x Mt
