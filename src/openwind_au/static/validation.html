Validation Scope

Validation checks run the normal terrain-profile and preliminary topographic screening workflow against manually selected representative Australian sites.

Validation is qualitative only. It does not prove AS/NZS 1170.2 compliance, wind-design accuracy, or suitability for a specific project.
